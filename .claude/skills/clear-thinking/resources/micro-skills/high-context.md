# 高上下文降噪微技能

本文件只在 `ROUTER.md` 判断问题存在高反思、高归因、持续分析但难行动，或需要区分结构信号、个人状态、身体/关系/行动边界时读取。

## 1. 痛感分流

- 适用处境：用户或材料中痛感很强，但不确定是结构问题还是个人状态。
- 输入：不适触发因素、事实证据、身体/关系/工作状态。
- 操作：分流为结构信号、个人状态、身体疲惫、关系问题、行动问题。
- 产出：该分析、休息、沟通、行动还是止损。
- 失效信号：用户只要具体执行，不需要情绪/结构判断。

## 2. 解释关闭

- 适用处境：持续分析但没有行动，或用理解替代切断。
- 输入：已解释内容、行动延迟、责任边界。
- 操作：判断继续解释是否服务行动；识别哪些责任不是本轮承担。
- 产出：停止分析或进入行动的条件。
- 失效信号：事实仍不足，关闭会导致误判。

## 3. 切断力

- 适用处境：需要拒绝、退出、止损、重构，而不是继续理解。
- 输入：继续投入收益、损耗、不可承受风险。
- 操作：判断到此为止的信号；设计低成本退出或边界。
- 产出：止损线和下一步。
- 失效信号：问题仍可通过低成本沟通或实验解决。
